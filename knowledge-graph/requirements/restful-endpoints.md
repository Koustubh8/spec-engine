# RESTful Endpoints

The API SHALL expose FastAPI endpoints following REST conventions:

- `GET /api/nodes` — list all nodes with filtering
- `GET /api/nodes/{slug}` — single node with edges
- `GET /api/edges` — list edges with filtering
- `POST /api/query/traverse` — graph traversal
- `POST /api/query/path` — path finding
- `POST /api/sync` — trigger sync
- `GET /api/stats` — dashboard stats

All endpoints SHALL return JSON with Pydantic response models (not raw dicts). All endpoints SHALL have OpenAPI documentation auto-generated by FastAPI.
