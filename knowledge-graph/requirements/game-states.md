# Game States

The game SHALL have states: START (title screen with instructions), PLAYING (main gameplay), GAME_OVER (final score), LEVEL_COMPLETE (boss defeated). START shows `ASCII SHOOTER` title and `PRESS SPACE`. GAME_OVER shows final score and `PRESS SPACE TO RESTART`.
