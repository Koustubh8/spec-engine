# Order Capture Workflow

The process of recording each candle order as it arrives via DM. A lightweight CLI replaces the current mental workflow.

Steps: DM arrives -> CLI records order -> payment screenshot confirmed -> shipment dispatched.
Weekly: pipeline run feeds data into forecasting.
