# Pixel Sprite Spec

All sprites generated programmatically using raylib GenImageColor + ImageDrawPixel. Player (20x28), ground tile (32x32), platform (32x16), enemies (18x16), coin (12x12), spike (32x32). Gradient sky background with cloud dots.
